# 支持 · Support

## 中文

- 使用问题、缺陷和建议：优先使用 GitHub Issues，并选择对应模板。
- 安全漏洞：请按 [SECURITY.md](SECURITY.md) 私下发送邮件。
- 其他联系：[asbacklight@gmail.com](mailto:asbacklight@gmail.com)。

提交问题时请注明版本、操作系统、CPU 架构和复现步骤，并对截图与日志脱敏。项目按维护者可用时间提供社区支持，不保证固定响应或解决时限。

## English

- Usage questions, bugs, and feature ideas: use GitHub Issues and select the matching template.
- Security vulnerabilities: email privately as described in [SECURITY.md](SECURITY.md).
- Other contact: [asbacklight@gmail.com](mailto:asbacklight@gmail.com).

Include the app version, operating system, CPU architecture, and reproduction steps. Redact screenshots and logs before uploading. Community support is provided as maintainer time permits, with no guaranteed response or resolution time.
