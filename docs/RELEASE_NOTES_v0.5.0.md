# 工位岛 v0.5.0 · Workday Island v0.5.0

## 中文

这是工位岛首个公开开源版本。它提供完整/精简双模式、可置顶桌面窗口、下班倒计时、天气、待办与持续提醒、今日已赚估算，以及结束后提醒休息的专注模式。界面支持中文、英文和跟随系统。

### 下载

- **macOS 12+**：下载 `Workday-Island-v0.5.0-macOS-universal.dmg`，同时支持 Apple Silicon 与 Intel。
- **Windows 10/11 x64**：下载 `Workday-Island-v0.5.0-windows-x64-Setup.exe`。
- `SHA256SUMS.txt` 可用于校验下载文件完整性。

安装包当前未使用商业代码签名证书，首次打开时操作系统可能显示安全提示。请确认下载地址为本仓库的 GitHub Release。

## English

This is the first public open-source release of Workday Island. It includes full and compact modes, an always-on-top desktop window, off-work countdown, weather, todos with persistent alerts, estimated earnings, and focus sessions that remind you to take a break. The interface supports Chinese, English, and system-language detection.

### Downloads

- **macOS 12+**: download `Workday-Island-v0.5.0-macOS-universal.dmg`; it supports both Apple Silicon and Intel.
- **Windows 10/11 x64**: download `Workday-Island-v0.5.0-windows-x64-Setup.exe`.
- Use `SHA256SUMS.txt` to verify downloaded files.

The packages are not yet signed with commercial code-signing certificates, so the operating system may show a security prompt on first launch. Verify that the download came from this repository's GitHub Release.
