# 工位岛隐私说明

[中文](PRIVACY.zh-CN.md) · [English](PRIVACY.md)

更新日期：2026-07-20

工位岛采用本地优先设计，不要求注册账号，也没有自有云端服务。

## 本机保存的数据

以下内容保存在用户配置目录中的 `WorkdayIsland/data.json`：

- 待办标题、备注、完成状态和提醒时间；
- 上下班时间、工作日、月薪与月计薪天数；
- 天气城市、界面语言、主题、货币符号、置顶、精简模式选项及上次调整的精简窗口尺寸；
- 最近一次成功获取的天气结果，仅用于断网时回退展示；
- 最近一次成功检查更新的时间，用于将自动检查频率限制为每天一次；
- 当前专注会话的开始、结束和运行状态。

应用不会主动上传这些数据。卸载程序不会自动删除数据文件，以便重新安装后继续使用；如需彻底清除，请在退出应用后手动删除 `WorkdayIsland` 配置目录。

## 联网行为

天气功能会把用户填写的城市名称发送给 Open-Meteo 地理编码服务，再使用解析出的经纬度查询当前天气。版本检查会访问本项目公开的 GitHub Releases API，读取版本号、更新说明和安装包地址；不会发送待办、薪资或其他本机业务数据。自动版本检查每天最多一次，也可在“关于”页手动触发。禁用网络不会影响待办、提醒、倒计时和专注功能。

## 系统能力

应用会使用系统通知、提示音和窗口置前能力来执行提醒。它没有遥测、行为分析、广告 SDK、账号登录、联系人/日历读取，也不会扫描其他文件。
