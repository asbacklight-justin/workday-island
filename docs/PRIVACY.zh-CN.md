# 工位岛隐私说明

[中文](PRIVACY.zh-CN.md) · [English](PRIVACY.md)

更新日期：2026-07-23

工位岛采用本地优先设计。待办等核心功能不要求注册账号；可选的实时聊天使用一键创建的匿名设备身份，不要求输入用户名或密码。

## 本机保存的数据

以下内容保存在用户配置目录中的 `WorkdayIsland/data.json`：

- 待办标题、备注、完成状态和提醒时间；
- 上下班时间、工作日、月薪与月计薪天数；
- 天气城市、界面语言、主题、货币符号、置顶、精简模式透明度与待办选项，以及上次调整的精简窗口尺寸；
- 最近一次成功获取的天气结果，仅用于断网时回退展示；
- 最近一次成功检查更新的时间，用于将自动检查频率限制为每天一次；
- 当前专注会话的开始、结束和运行状态。
- 实时聊天的匿名用户 ID、设备 ID、公开凭据 ID、显示昵称，以及最多 500 条最近实时消息和窗口互动记录。

Ed25519 设备私钥不会写入上述 JSON：macOS 使用系统钥匙串，Windows 使用 DPAPI 加密后保存在用户配置目录。应用不会上传待办、薪资、工作时间、专注记录或设备私钥。卸载程序不会自动删除数据文件，以便重新安装后继续使用；如需彻底清除，请在退出应用后手动删除 `WorkdayIsland` 配置目录，并在应用内重置实时身份。

## 联网行为

天气功能会把用户填写的城市名称发送给 Open-Meteo 地理编码服务，再使用解析出的经纬度查询当前天气。版本检查会访问本项目公开的 GitHub Releases API，读取版本号、更新说明和安装包地址；不会发送待办、薪资或其他本机业务数据。自动版本检查每天最多一次，也可在“关于”页手动触发。禁用网络不会影响待办、提醒、倒计时和专注功能。

只有用户点击“一键上线”后，应用才会连接 `admin.asbacklight.cn`：首次使用会发送昵称、设备 ID、设备名称和 Ed25519 公钥来创建匿名设备身份；上线时使用设备签名认证；聊天消息、发送者/接收者用户 ID、发送时间、送达与已读状态，以及抖动/闪烁指令会经过实时服务并可能由服务端持久化，以便对方离线后补发。用户可随时下线或重置本机实时身份。禁用网络不会影响聊天以外的核心功能。

## 系统能力

应用会使用系统通知、提示音和窗口置前能力来执行提醒与用户主动发送的窗口互动。它没有遥测、行为分析、广告 SDK、用户名密码登录、联系人/日历读取，也不会扫描其他文件。
