# 安全政策 · Security Policy

## 中文

当前维护版本为 `0.5.x`。发现可能导致本地数据泄露、任意代码执行、安装包篡改或不安全网络访问的问题时，请不要创建公开 Issue。

请发送邮件至 [asbacklight@gmail.com](mailto:asbacklight@gmail.com)，主题注明 `Workday Island Security`，并包含：受影响版本和平台、复现步骤、影响说明以及可行的缓解建议。我们会尽量在 7 天内确认收到，并在验证、修复和发布安全更新后协调披露时间。请不要在测试中访问他人设备或数据。

只有 GitHub 仓库 Release 页面中的文件属于官方发布。安装包哈希会随 Release 提供；代码签名上线前，建议下载后核对 SHA-256。

## English

The currently maintained line is `0.5.x`. Do not open a public issue for a suspected vulnerability involving local data exposure, arbitrary code execution, package tampering, or unsafe network access.

Email [asbacklight@gmail.com](mailto:asbacklight@gmail.com) with the subject `Workday Island Security`. Include the affected version and platform, reproduction steps, impact, and any suggested mitigation. We aim to acknowledge reports within seven days and coordinate disclosure after validation, a fix, and a security release. Do not access another person's device or data while testing.

Only files on this repository's GitHub Releases page are official distributions. Release notes include package hashes; verify SHA-256 values until code signing is available.
